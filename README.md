# RAG PDF Chatbot

A Retrieval-Augmented Generation (RAG) based chatbot that allows users to upload PDFs and ask questions. The system extracts text, chunks it, embeds content, stores vectors, and retrieves the most relevant information to answer queries accurately.

## Features

Upload and process PDF documents

Text chunking + embeddings for accurate retrieval

Vector search for relevant context

Answer generation grounded in documents

Minimal hallucination response style

Works with multiple PDFs
